# Monitoring Dashboard - DevOps Assignment

A complete microservices-based monitoring dashboard with FastAPI backend, React frontend, Docker, Kubernetes, and CI/CD.

## Features
- FastAPI backend exposing `/metrics`
- React dashboard with real-time graphs
- Docker & Kubernetes deployment
- GitHub Actions CI/CD
- Ready to run locally using Docker Compose

## How to Run
```
docker-compose up --build
```
